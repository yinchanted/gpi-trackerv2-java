# gpi tracker v2 java client [![Build Status](https://travis-ci.com/swiftinc/gpi-tracker-v2-java.svg?branch=master)](https://travis-ci.com/swiftinc/gpi-tracker-v2-java)

## Overview

The client classes are generated by swagger-codegen, we added unit tests, example tests and a helper class to help you consume the GPI tracker V2 API

## Installation

To install the API client library to your local Maven repository, simply execute:

```shell
mvn install
```

### Maven users

Add this dependency to your project's POM:

```xml
<dependency>
    <groupId>com.swift.gpi.tracker</groupId>
    <artifactId>v2sdk</artifactId>
    <version>1.0.0</version>
    <scope>compile</scope>
</dependency>
```

### Others

At first generate the JAR by executing:

    mvn package

Then manually install the following JARs:

* target/v1sdk-1.0.0.jar
